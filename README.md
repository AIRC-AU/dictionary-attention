# Neural Machine Translation System

```
usage: translate.py [-h] [--train FILE] [--score FILE] src_lang tgt_lang [input]

positional arguments:
  src_lang      source language
  tgt_lang      target language
  input         input string for translation

optional arguments:
  -h, --help    show this help message and exit
  --train FILE  train model
  --score FILE  score model
```
